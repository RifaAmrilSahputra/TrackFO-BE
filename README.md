### 📝 Deskripsi
**TrackFO Backend** merupakan pusat data dan logika utama berbasis **REST API** yang dirancang khusus untuk mendukung ekosistem aplikasi **TrackFO Mobile**. Dibangun dengan arsitektur yang fokus pada performa dan skalabilitas.

---

### 🚧 Status Proyek
> [!IMPORTANT]
> Proyek ini sedang dalam tahap pengembangan aktif pada sisi backend.  
> Modul autentikasi dan manajemen user telah selesai diimplementasikan dengan standar keamanan (JWT, rate limiting, role-based access control).  
>  
> Saat ini pengembangan difokuskan pada modul inti sistem, yaitu:
> - Manajemen gangguan
> - Assignment teknisi (manual & otomatis)
> - Tracking lokasi teknisi (real-time)
> - Laporan penyelesaian pekerjaan  
>  
> Struktur API dan skema database pada modul inti masih dapat mengalami perubahan seiring pengembangan.
